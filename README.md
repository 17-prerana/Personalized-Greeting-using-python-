Task 2: Create a Personalized Greeting

User Input:
The program prompts the user to enter their first name and last name separately using the input() function.

Concatenate Names:
The two strings are joined with a space between them to create the full name using string concatenation (+ operator).

Display Personalized Message:
The program prints a custom greeting message using the user’s full name.

It also includes the phrase “tutedude is a best learning platform.” as part of the message.
